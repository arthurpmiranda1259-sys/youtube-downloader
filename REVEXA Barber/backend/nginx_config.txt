# Configuração Nginx para REVEXA Barber

Adicione estas linhas no arquivo de configuração do site:

```nginx
# Proxy para WhatsApp (temporário - PHP)
location /whatsapp/ {
    rewrite ^/whatsapp/(.*) /revexa_sistemas/Sistemas/Revexa_Barber/whatsapp_proxy.php?endpoint=$1 last;
}

# Quando configurar o servidor Node.js, substitua por:
# location /whatsapp/ {
#     proxy_pass http://localhost:3001/;
#     proxy_http_version 1.1;
#     proxy_set_header Upgrade $http_upgrade;
#     proxy_set_header Connection 'upgrade';
#     proxy_set_header Host $host;
#     proxy_cache_bypass $http_upgrade;
# }
```

Depois execute:
```bash
sudo nginx -t
sudo systemctl reload nginx
```
